Angular 2.1 starter pack

Setup for Angular 2.1 with Webpack & materialize

Set up an Angular 2.1 project with Webpack, Materialize and unit testing.
Clone the repository: 

git clone https://github.com/dennieru/semaphore-ng2-webpack.git

Install all dependecies: 

npm install

And run: 

npm start

Starter pack project should be available at: http://localhost:7000/
